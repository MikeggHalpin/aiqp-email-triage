🔐 Connect Your Outlook

AIQP Email Triage uses the Microsoft Graph API to read your inbox. You'll need an Azure App Registration with the Mail.Read and Mail.Send delegated permissions.

From your Azure AD App Registration → Overview → Application (client) ID
Your Azure tenant ID, or type "common" for any Microsoft account
⚙️ One-time setup: You'll need an Azure App Registration with redirect URI set to https://claude.ai (or your hosted URL) and Mail.Read, Mail.Send, Mail.ReadWrite permissions granted.

💡 Demo mode: Leave Client ID blank and click the button to load sample emails.
